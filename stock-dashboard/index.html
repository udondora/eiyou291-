    ④ チャート（TradingView）

    チャートはTradingViewの公式埋め込みウィジェット。判定パネルは取得したOHLCVから本ツールが独自計算しています。
    判定ロジック（要約）
    Trend＝価格>9EMA>21EMA の段階・MA上向き・RS良好で OK。
    Entry＝Trendが OK かつ 過熱(21EMAからの乖離)が小さく出来高が伴う押し目/ブレイクで OK、伸び切りは 待機。
    利確＝MAからのATR乖離が過大・%乖離が過大などの過熱で OK(利確検討)。
    行動＝上記を統合（買い候補／保有／待機／利確）。閾値はコード先頭の CONFIG で調整可能。
