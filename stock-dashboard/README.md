# 株価判断ダッシュボード（TradingView連携 / トレンドテンプレート）

銘柄の日足データから **Minervini風トレンドテンプレート**を機械的に判定し、
`Trend / Entry / 利確 / 行動` を **OK / NG / 待機** で可視化する単一ファイルの Web アプリです。
ビルド不要・`index.html` をブラウザで開くだけで動作します。

> ⚠️ 投資助言ではありません。判定の最終責任はご自身にあります。

## 特徴

- **自前の価格チャート** — 取得した OHLCV からローソク足・移動平均(20/50/75/100/150/200)・出来高・RSI(14) を独自描画（v41）。判定パネルも同じ OHLCV から独自計算。
- **3つのデータ取得元** — Stooq（無料・キー不要）/ Twelve Data（無料APIキー）/ CSV貼付（TradingViewのエクスポートCSVをそのまま）。
- **判定テンプレート再現** — 9EMA>21EMA の段階、主要MA上・上向き、RS Rating（近似）、MAからのATR乖離倍・%乖離、出来高倍率、半値返し、過熱判定。
- **トレンドテンプレート8条件チェック（v2）** — Minervini の8条件を ✓/×/— で一覧表示し「○/8 条件」を集計。
- **自然文アドバイス（v2）** — 判定に応じて「買い候補／保有・待機／利確検討／様子見」の解説とリスク管理の目安を文章で提示。
- **視覚ゲージ＋指標追加（v2）** — RS Rating・21EMA乖離（過熱度）・52週レンジ内の位置・RSI(14)・出来高倍率をバーで可視化。RSI と「52週安値からの上昇率」も追加。
- **価格レベル＆ポジションサイジング（v3）** — 損切り目安（スイング安値／ATR×2の高い方）、想定リスク%、利確目標 2R/3R、R:R を表示。口座資金と1回のリスク%を入れると**推奨株数・投入額**を自動計算。
- **最近の銘柄＆状態記憶（v3）** — 直近に判定した銘柄をワンタップで再判定。最後の銘柄・口座資金も記憶。
- **ウォッチリスト・スクリーニング（v7/v8）** — 複数銘柄（最大30）をまとめて判定し、**トレンドテンプレ達成数（○/8）・RS・21EMA乖離**で並べ替え。行動バッジ表示、銘柄タップで単体判定へジャンプ。SPYキャッシュ＋8回/分レート制御で全件取得。
- **エントリー厳しさプリセット＋押し目認識（v9）** — 買い判定を「緩い／標準／厳しい」で切替。強いトレンドで21EMA近辺まで調整した銘柄は**押し目買い候補**として検出（出来高不問）。
- **買い候補フィルター（ON/OFF式・v12/v13）** — ①地合い（SPYが上昇トレンド時のみ）②健全な押し目のみ ③ブレイクは52週高値圏のみ ④RS≥70必須 ⑤損切り幅が大きい銘柄を除外。各チェックボックスで個別にON/OFF（不要なら"キル"）でき、**表示中の単体判定もスクリーニング表も再取得なしで即再評価**。
- **総合スコア＆自動ランキング（v14）** — RS・条件達成数・セットアップ・地合いを合成した0-100スコアで「今買うならどれ」を上位表示。単体判定にもスコアバー。
- **新高値ブレイク接近（v14）** — 52週高値−3%以内を🔥フラグ＋スコア加点。
- **流動性・週足フィルター（v14）** — 平均売買代金が小さい銘柄を除外／30週線(≒150日)の上＋上向き(ステージ2)のみ買い候補（ON/OFF）。
- **ベンチマーク選択（v14/v22）** — RS・地合いの基準を RSP（均等加重・IBD近似／既定）/ SPY / QQQ で切替。RSPは「平均的な銘柄」に近く、IBDのRS Rating（全銘柄パーセンタイル）に最も整合。
- **スクリーニング結果の保存＆CSV出力（v14）** — 次回起動時に前回結果を復元、CSVダウンロード可。
- **PWA対応（v14）** — ホーム画面に追加でアプリのように起動、オフラインでも前回結果を表示。文字サイズ(A-/A+)切替。
- **更新まわり＆ナビ改善（v15）** — 新バージョンを自動検知して「🆕更新」バーで即最新化、🔄手動更新ボタン（キャッシュ全消去）、トップへ戻るフローティングボタン、買い候補フィルターは折りたたみ式（有効数を表示）。
- **レイアウト/ロジック見直し（v16）** — 下部の詳細カード（テンプレ/ゲージ/指標）を折りたたみ式にして縦を短縮、スコアを強調＋セクション区切りを追加。判定ロジックの不具合（trend判定で21EMAの向きが無視されていた）を修正し、上昇トレンド＝段階＋主要MA上＋21EMA上向きに厳密化。
- **バックテスト（v17）** — 過去の各日まで（未来を見ない）で判定ロジックを再実行し、買い候補→翌日始値エントリー、損切り/利確(R)/トレンド崩れで手仕舞いを再現。勝率・期待値(R)・プロフィットファクター・最大DD・合計リターン・Buy&Hold比較・トレード履歴を表示。損切り方式・利確R・コストを調整可。長期データを自動取得。※単一銘柄・限定期間のイン・サンプル検証（参考値）。
- **プール検証＆OOS分割（v18）** — ウォッチリスト全体の取引を合算して統計的信頼度を上げる「プール・バックテスト」。さらに前半(イン・サンプル)と後半(アウト・オブ・サンプル＝未知データ)に分けて成績を比較し、過剰最適化（カーブフィッティング）でないかを検証。
- **売却・保有判断（v19）** — 保有者向けの出口判断：トレンド崩れ/50日線割れ/スイング安値割れ→「売り」、21EMA割れ/分配の兆候→「警戒（縮小）」、過熱→「一部利確」、健全→「継続保有」。支持線（21EMA→50日線→スイング安値）の割れ状況と提案損切りを表示。取得単価を入れると含み損益＋損切りライン到達も判定。
- **損切り基準の選択（v21）** — 損切りラインを「アプリ推奨（スイング/ATRの高い方）／スイング安値／ATR×倍率／21EMA直下／固定%」から選択。価格レベル・想定リスク・利確目標・推奨株数・売却カードの提案損切り・スクリーニングのリスク判定すべてに即反映。
- **トレンド判定の精緻化（v20）** — 上昇トレンドを「価格>21EMA＋9EMA>21EMA」に定義（9EMAの一時割れ＝押し目はトレンド維持）。強い銘柄の浅い押し目を「様子見/売り」と誤判定しないよう修正。
- **IBD RS 手入力（v23）** — IBD等の正確なRS Rating値を入力すると、近似値の代わりにその実値を使用。トレンドテンプレート条件8（RS≥70）の合否・RS表示・買い候補フィルタ・スコアに即反映（再取得なし）。空欄なら従来の近似値。
- **スコアの内訳表示＆配分調整（v24）** — 総合スコアの計算根拠（RS×重み＋8条件達成率×重み＋セットアップ×重み、地合い×0.85・新高値+3）を内訳表示。RS/8条件/セットアップの配分(%)を自分で調整可（自動正規化）。
- **最低スコア条件＆重み自動最適化（v25）** — バックテストに「最低スコア以上の買い候補だけ建てる」条件を追加。さらにRS/8条件/セットアップの配分を総当たり（48通り）し、**OOS（未知データ）期待値が最大の配分**を提示（重みに依存しない計算を事前1回キャッシュして高速化）。結果行タップで配分を即適用。
- **表示順の最適化（v26/v27）** — ワークフロー順に配置：入力 → ウォッチリスト・スクリーニング → トレンド/買い・売り判断 → 価格レベル → 詳細、フィルター/バックテスト等のツールは下部。無駄なスクロールを削減。
- **判定の理由を明確化（v28/v29）** — 判定チップの下に「トレンド／エントリー／利確」それぞれの理由を1行ずつ色付きで表示。例: トレンド＝上昇配列完成、エントリー＝伸び切りで押し目待ち、利確＝過熱なし保有OK。「待機」の意味と"何を待つか"が一目で分かる。
- **指標・条件の平易説明（v30）** — トレンドテンプレート8条件の×に「なぜ×か」のヒントを表示。主要指標（RS/乖離/RSI/52週位置/ボラ等）に「非常に強い／過熱／買われ過ぎ／新高値圏🔥」等の意味を注記。
- **ウォッチリスト即時保存（v31）** — 銘柄の追加・削除を編集した瞬間に保存（スクリーニング実行を待たない）。
- **価格レベルの平易化（v32）** — 損切り/1R/利確2R・3R/推奨株数を①〜④の番号と1行説明で解説。利確目標に現値からの+%を併記。
- **用語ヘルプ（v33）** — トレンドテンプレート/RS/R/損切り/ポジションサイジング/21EMA/スイング安値/押し目/ATR/地合い/過熱/スコア/OOS など14語を平易に解説する折りたたみ用語集。
- **保有管理・ミニポートフォリオ（v34）** — 保有銘柄＋取得単価（株数任意）を登録し、含み損益（%・額）・損切りライン到達⚠️・売り/警戒/一部利確/継続保有の判定を一覧でまとめて確認。「一括更新」で全銘柄を再取得・再判定。銘柄タップで単体判定へジャンプ。localStorageに保存。
- **テーマ・トラッカー（v35／銘柄構成 v37で刷新）** — 主要10テーマ（AI半導体／半導体製造装置／メモリ・ストレージ／光通信・ネットワーク／AIインフラ・データセンター／クラウド・AIソフト／電力・原子力／燃料電池・クリーンエネ／宇宙・防衛／量子コンピュータ）の構成銘柄をまとめて取得し、**平均騰落率でランク付け**。テーマ内で20日線の上にある銘柄数・新高値接近🔥数・ミニ推移グラフを表示。期間（1週/1ヶ月/3ヶ月）切替、テーマ行タップで構成銘柄を騰落率順に展開（銘柄タップで単体判定へ）。「リーダー銘柄はリーダーテーマから」の発想。
- **ブレイク・サマリー（v36）** — テーマ・トラッカーで取得済みの全銘柄を横断し、市場の勢いを集計：52週高値更新数・52週安値更新数・移動平均(20/50/200)を上抜け/下抜けした銘柄数（直近5営業日のクロス）・出来高急増(平均1.5倍)数。MA長さ・方向は切替式（再取得なし）。
- **テーマ・トラッカーの見やすさ強化（v38）** — カード型レイアウトに刷新。各テーマに塗りつぶしスパークライン（直近6ヶ月・推移レンジ%付き）・期間騰落率・1日変化率・出来高変化率（直近20日平均÷その前20日−1）・20日線上数・🔥数を表示。期間ボタンを1日/1週/1ヶ月/3ヶ月/6ヶ月に拡張（再取得なしで並べ替え）。構成銘柄の展開に会社名と1日変化率を追加。
- **ブレイク銘柄の内訳＆データ保存（v39）** — ブレイク・サマリーの各数字（52週高値/安値更新・MAブレイク・出来高急増）をタップすると該当銘柄を一覧表示（銘柄タップで単体判定へ）。テーマ取得データはブラウザに保存し、次回起動時は前回データを即表示。再取得は「テーマ分析」押下時に確認ダイアログ（OK＝最新取得／キャンセル＝前回表示）で選択。取得時刻も表示。
- **保存とデータ更新確認をスクリーニング/保有管理にも拡張（v40）** — テーマ・トラッカーと同じく、スクリーニングと保有管理も取得データをブラウザに保存。次回起動時は前回結果を即表示（取得時刻つき）。再取得は実行ボタン押下時の確認ダイアログ（OK＝最新取得／キャンセル＝前回表示）で選択。スクリーニングは系列も保存するため、再読込後もフィルター変更が再取得なしで即反映。
- **自前の価格チャート＋ブレイク銘柄ジャンプ（v41）** — 取得したOHLCVからローソク足・移動平均(20/50/75/100/150/200)・出来高・RSI(14)を自前描画。ヘッダーに「半年 +X% ・ 1日 +Y% ・ 終値: 日付」を表示し、期間ボタン（1日/5日/10日/1ヶ月/2ヶ月/3ヶ月/6ヶ月/12ヶ月）は「表示する騰落率の期間」を選ぶ方式（v44）。ヘッダーは常に「選んだ期間の騰落率 ＋ 1日の騰落率」を表示し、チャートは最低3ヶ月の日足（6ヶ月/12ヶ月選択で伸長）。ブレイク・サマリーの該当銘柄チップはチップ全体タップで個別判定へジャンプ。
- **TradingView 埋め込みチャートを削除（v42）** — 自前チャート（v41）に一本化。外部ウィジェットの読み込みをなくして軽量化・オフライン対応を強化（CSV は引き続き TradingView エクスポート形式に対応）。
- **テーマ・トラッカーの見た目を刷新（v45）** — 参考アプリ風のカード：緑のランク丸＋緑の左ライン、カテゴリのカラーバッジ、1年間推移スパークライン（上下に最大/最小%・ゼロ基準線）、大きな騰落率＋1日＋出来高変化率。列見出し「1年間推移 / Nヶ月騰落率」。既定期間を3ヶ月に。展開時の構成銘柄も同じカード調（ランク丸＋ティッカー＋会社名＋1年間推移＋騰落率、タップで個別判定へ）。
- **全体の連動・統一（v47）** — 画面上部に固定の**セクション・タブ**（判定/テーマ/スクリーニング/保有/チャート）でジャンプ、スクロールに応じて現在地をハイライト。**期間セレクタを共通化**し、テーマ一覧の騰落率とチャートの騰落率を1か所で切替（1日〜1年）。個別判定から「💼保有に追加」「🔎ウォッチに追加」で各機能へ**ワンタップ導線**。
- **見た目の全面統一（v48）** — スクリーニング結果と保有管理を表からカードへ刷新（スコア/損益バッジ＋ティッカー＋会社名＋行動・判定バッジ＋主要指標、タップで個別判定へ）。スクリーニングは並べ替えチップ（★スコア/RS/条件/乖離/銘柄）。全セクションの見出しにアクセントバーを付け、テーマ一覧と同じデザイン言語に統一。

## 使い方

1. `index.html` をブラウザで開く。
2. 銘柄シンボル（例 `NBIS`）と取引所（チャート用、例 `NASDAQ`）を入力。
3. データ取得元を選ぶ：
   - **Stooq**: キー不要。CORS の都合で取得できない場合は CSV貼付を利用。
   - **Twelve Data**: [twelvedata.com](https://twelvedata.com) で無料キーを取得し入力。
   - **CSV貼付**: TradingView のチャートからエクスポートした CSV を貼り付け。
4. 「判定する」を押すと、総合判定・主要指標・チャートが表示されます。

## 判定ロジック（要約）

| 項目 | 条件 |
|------|------|
| **Trend** | 価格 > 9EMA > 21EMA の段階、主要MA上＆21EMA上向き、RS良好 → `OK` |
| **Entry** | Trend が OK かつ過熱でなく（21EMAからの%乖離が小）出来高が伴う → `OK`、伸び切りは `待機` |
| **利確** | MAからのATR乖離倍が過大／%乖離が過大などの過熱 → `OK`（利確検討） |
| **行動** | 上記を統合：`買い候補 / 保有・待機 / 様子見 / 利確検討` |

閾値は `index.html` 先頭の `CONFIG` オブジェクトで調整できます
（過熱閾値、出来高倍率、RS良好ライン、エントリー許容乖離など）。

### 主要指標
- **RS Rating（近似）**: SPY 比の複数期間（約3/6/9/12か月）加重を 1–99 にマップした IBD 風の近似値。
- **% Gain From MA / ATR% Multiple From MA**: 21EMA からの乖離（%／ATR倍）。過熱・伸び切りの目安。
- **ATR%(14) / 出来高倍率 / 半値返し / 52週高値からの距離**。

## 制限・注意

- Stooq・Twelve Data の無料枠やレート制限、ブラウザの CORS 制約により取得が失敗する場合があります。その際は **CSV貼付**が確実です。
- RS Rating は本来の全銘柄パーセンタイルではなく、SPY 比からの**近似**です。
