# eiyou291

管理栄養士国家試験（午前）対策の **291問 問題集アプリ**。
HTML 単一ファイルで動作し、ブラウザで開くだけで使えます。スマートフォン・iPad にも対応し、ホーム画面に追加すればアプリのように使えます。

## 特徴

- **291問を収録** — 各問に選択肢・正誤判定・詳解・直前暗記メモ・ひっかけ対策付き。
- **問題文の読みやすさ改善（v35→v36）** — 元データ由来で文の途中に入っていた不自然な改行を整形。v36では「文末（。）や表・資料の区切りは残し、文の途中の改行だけをつなぐ」賢い処理に変更し、表を含む問題（資料つき）でも崩れないようにした（全291問対応）。
- **選択肢のトグル開閉（v34）** — 選んだ選択肢をもう一度タップすると、解説を閉じて未選択の状態に戻る（記録も取り消し）。問題の頭へ戻ってすっきり解き直せる。
- **問題の区切りを明確化（v33）** — 各問カードに色付きの左ライン（未挑戦＝出題回の色／正解＝緑／不正解＝赤）・余白拡大・影・見出し下の区切り線を追加。スクロール中も1問ずつの境目と解答状況が一目で分かる。
- **品質改善・ヘルプ追加（v32）** — 医療内容のファクトチェック実施（図解10点はすべて正確。食中毒のO157に「生体内毒素型」の注記を追加）。テスト（模試）中に正誤の色が見えてしまう不具合や、採点後のキー操作で進捗が書き換わる不具合など複数のバグを修正。冒頭に「❓ はじめての方へ（使い方・色の意味）」を追加し、緑＝正解／赤＝不正解／青＝選択中・操作／黄＝注意 の**色の凡例**を明示。
- **操作パネルのコンパクト化（v31）** — 学習トラッカーを大幅にスリム化（高さ約半分）。常時表示は「正答率・正解/不正解/残り・続きから・検索・表示モード」だけにし、テスト/集中/文字サイズ/進捗データ等は「🧭 学習メニュー・ツール」に集約。出題範囲（回・難易度・科目）は折りたたみ化。画面下の固定チップを廃止し、指標表記も簡潔化。**出題回/難易度/科目フィルタが効かなかった既存不具合も修正**。
- **テスト（模試）モード（v30）** — 現在の表示対象からランダムに 10／20／30／全問を出題。解答中は答えを隠し、「採点する」で得点・正答率を表示。終了後は「間違いだけ復習」へ。結果は学習トラッカーにも反映。
- **解答後に解説を自動表示＋「次の問題へ」（v30）** — 選択肢を押すと解説が自動で開き、各問に「次の問題へ →」ボタンを表示（⚙ツールでオン/オフ）。
- **キーボード操作（v30）** — `1`〜`5` で現在の問題に解答、`←`／`→` で前後の問題へ移動（iPad＋キーボード／PC）。
- **1問ずつ集中モード（v29）** — 長いスクロールをやめ、1問だけ大きく表示→「次へ／前へ」で進む。現在のフィルタ・表示モードに連動。スマホ・スキマ時間の集中学習に。
- **★ブックマーク（v29）** — 気になる問題に★を付け、「★だけ」表示で見直し。不正解とは別管理。
- **続きから再開（v29）** — 最後に解答した／集中モードで見ていた問題へワンタップで戻る（再読込・端末再起動後も保持）。
- **進捗のバックアップ・復元（v29）** — 進捗・★・設定を JSON で書き出し／読み込み。端末を変えても引き継げ、配った相手にも初期データを渡せる。
- **画面構成・配色の刷新（v28）** — ファーストビューを短縮（導入文を整理し、トラッカー直下にすぐ問題が来るレイアウト）。二次的な操作（出題・文字サイズ・科目別）は「⚙ ツール」に集約。配色は cyan=操作／green=進捗・正解／amber・red=注意 に役割を整理し、コントラストを向上。科目別バーは未挑戦をグレーで区別。
- **図解で覚える（v27）** — 高頻出テーマを教科書準拠の図解で整理（食事摂取基準の5指標カーブ、感度・特異度の2×2、ホルモンの作用方向、アレルギーⅠ〜Ⅳ型、周産期・新生児・乳児死亡の区分、疫学の2×2と式、社会保障の4本柱、ビタミン、食中毒、WBGTの運動指針 ほか全10点）。
- **選択肢○✕マップ（v27）** — 各問に、選択肢ごとの ○／✕ と「正しくは」を一覧化した整理ビューを自動生成。「📊 選択肢マップ」で全問を一括開閉。
- **科目別の弱点分析（v27）** — 学習トラッカーに科目別の正答率バーを表示。苦手分野（正答率が低い科目）が一目で分かる。
- **文字サイズ切替（v27）** — A−／A＋で問題文の表示倍率を4段階に変更（設定はブラウザに保存）。
- **ランダム出題・シャッフル（v27）** — 表示順をランダム化／元の順に戻す。本番の順不同に慣れる用。
- **印刷・PDF用レイアウト（v27）** — 操作UIを隠し、問題＋選択肢＋○✕マップを白背景で整形印刷（紙で配る・PDF保存用）。
- **学習トラッカー（v26）** — 解答すると正解／不正解を自動記録し、**正答率・正解数・不正解数・未挑戦数・最大連続正解**をダッシュボード表示。記録はブラウザ（localStorage）に保存され、次回起動時に**選んだ選択肢まで復元**します。
- **復習モード（v26）** — 「不正解だけ」「未挑戦だけ」に絞って表示。間違えた問題の解き直し・残りの消化がワンタップ。
- **キーワード検索（v26）** — 設問・テーマ・選択肢の文言から問題を即時フィルタ（例：インスリン／周産期死亡率／WBGT）。
- **iPad / iPhone アプリ化（v26 / PWA）** — `manifest.webmanifest`・Service Worker・アイコンを同梱。Safari の「ホーム画面に追加」で全画面起動、オフラインでも動作。新バージョンは「🆕更新」バーで通知。
- **3軸フィルタ** — 出題回・難易度・科目で絞り込み可能（検索・復習モードと併用可）。
- **学習フロー** — 「問題を読む → 選択肢を押す → 解説を見る」のシンプルな導線。
- **モバイル最適化** — ダークテーマ、固定ヘッダ、`viewport-fit=cover` 対応。
- **JS無効でも動く** — フィルタや解説の開閉は CSS（ラジオ/チェックボックス）で実装。トラッカー・検索・復習・PWA のみ JavaScript で追加しており、JS が無効でも問題集としてそのまま使えます。

## 使い方

1. リポジトリをクローン、または HTML ファイルを直接ダウンロードします。
2. 利用したいファイル（推奨: `eiyou291_v36.html`）をブラウザで開きます。
3. 画面上部のフィルタ・検索・表示モードで出題範囲を絞り込みます。
4. 問題を読み、選択肢を押すと正誤と詳しい解説が表示され、学習トラッカーに記録されます。

> サーバーやビルドは不要です。ファイルをダブルクリックするだけで動作します。

### iPad / iPhone でアプリとして使う（PWA）

進捗の保存やオフライン動作・ホーム画面追加を完全に活かすには、`file://` で直接開くのではなく **HTTP(S) で配信**してください（Service Worker は `https`/`localhost` でのみ動作）。

- **手軽に試す（ローカル配信）**: リポジトリのルートで `python3 -m http.server` を実行し、`http://<PCのIP>:8000/` を iPad の Safari で開く。
- **公開する**: GitHub Pages 等にこのリポジトリを配信し、ルートURL（`index.html`）を開く。
- **ホーム画面に追加**: Safari の共有メニュー →「ホーム画面に追加」。全画面（standalone）で起動し、オフラインでも動作します。新バージョンを配信すると「🆕更新」バーが出るので、押すと最新化されます。
