# eiyou291

管理栄養士国家試験（午前）対策の **291問 問題集アプリ**。
JavaScript を一切使わず、HTML と CSS だけで動作する単一ファイル構成です。
ブラウザで HTML を開くだけで利用でき、スマートフォンでもそのまま使えます。

## 特徴

- **JSなしの安定動作** — フィルタや解説の開閉はすべて CSS（ラジオ/チェックボックス）で実装。ネット接続不要、ローカルで完結。
- **291問を収録** — 各問に選択肢・正誤判定・詳解・直前暗記メモ・ひっかけ対策付き。
- **3軸フィルタ** — 出題回・難易度・科目で絞り込み可能。
- **学習フロー** — 「問題を読む → 選択肢を押す → 解説を見る」のシンプルな導線。
- **モバイル最適化** — ダークテーマ、固定ヘッダ、`viewport-fit=cover` 対応。

## ファイル構成

最新版をルートに置き、過去バージョンは `archive/` にまとめています。
通常は **`eiyou291_v25.html`** を利用してください。

```
.
├── eiyou291_v25.html              # 最新・推奨（v25 講師メモ強化版）
└── archive/                       # 過去バージョン
    ├── eiyou291_detail_final_v17.html  # v17 判定安定版
    └── eiyou291_detail_v7.html         # v7 正答チェック・直前暗記強化版
```

| ファイル | バージョン | 説明 |
|----------|-----------|------|
| `eiyou291_v25.html` | v25 | 講師メモ強化版（最新・推奨） |
| `archive/eiyou291_detail_final_v17.html` | v17 | 判定安定版 |
| `archive/eiyou291_detail_v7.html` | v7 | 正答チェック・直前暗記強化版 |

## 使い方

1. リポジトリをクローン、または HTML ファイルを直接ダウンロードします。
2. 利用したいファイル（推奨: `eiyou291_v25.html`）をブラウザで開きます。
3. 画面上部のフィルタで **出題回 / 難易度 / 科目** を選び、出題範囲を絞り込みます。
4. 問題を読み、選択肢を押すと正誤と詳しい解説が表示されます。

> サーバーやビルドは不要です。ファイルをダブルクリックするだけで動作します。
